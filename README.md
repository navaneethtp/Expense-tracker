# Expense Tracker – Personal Finance Analytics Dashboard

A full-stack personal finance management web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) that helps users securely manage income, expenses, and financial activities through authentication, personalized dashboards, profile management, interactive analytics, and downloadable financial reports.

This platform is designed to simplify money management by combining secure account systems, transaction tracking, visual insights, XLSX export, emoji-based categorization, and responsive modern design into one scalable solution.

# Project Overview

Expense Tracker is a secure and responsive personal finance platform that enables users to register, log in, upload profile photos, manage financial transactions, visualize income and expenses, and export data for offline reporting.

The application focuses on financial awareness, personalization, and practical financial record management while enhancing user experience through emoji-based income and expense categorization, dark mode, and seamless usability across desktop, tablet, and mobile devices.

# Installation & Setup

## Clone the Repository
git clone https://github.com/navaneethtp/Expense-tracker.git<br>
cd Expense-tracker<br>
## Install Frontend Dependencies
cd client<br>
npm install<br>
npm run dev<br>
## Install Backend Dependencies
cd server<br>
npm install<br>
npm run dev<br>
## Environment Variables
MONGO_URI=your_mongodb_connection_string<br>
PORT=5000<br>
JWT_SECRET=your_jwt_secret<br>
